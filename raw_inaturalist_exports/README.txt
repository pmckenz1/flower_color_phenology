This directory contains raw exports from iNaturalist using their query and export tool (https://www.inaturalist.org/observations/export). There were twelve total files, since the maximum observation number per file is capped at 200,000. 

Each file contains, within some lat/lon bounded box, all of the angiosperm observations annotated as "flowering" AND that are of research grade AND that have open geoprivacy AND that have photos.

Going down this README we are sliding our windows east -> west across central North America across the individual exports, keeping the latitude boundaries the same for all queries but shifting the longitude boundaries, making them as big as possible at each step without going over 200,000 observations for the export.
